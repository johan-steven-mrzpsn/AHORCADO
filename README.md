# Proyecto Integrador – Juego del Ahorcado

- Nombre del estudiante: Johan Steven Quinapallo Tituaña
- Lógica de Programación
- Juego del Ahorcado

## Objetivo del programa
Desarrollar un programa en Python que implemente el juego del Ahorcado, aplicando los conceptos básicos de lógica de programación vistos en la asignatura.

## Funcionalidades principales del código
- Selección aleatoria de una palabra.
- Solicitud de letras al usuario.
- Validación de la entrada del usuario.
- Registro de letras correctas e incorrectas.
- Control del número de intentos.
- Mensaje final indicando si el jugador gana o pierde.
